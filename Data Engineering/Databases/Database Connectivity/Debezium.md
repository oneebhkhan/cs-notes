Support for the following YugabyteDB data types will be enabled in future releases:
	-   BOX
	-   CIRCLE
	-   LINE
	-   LSEG
	-   PATH
	-   PG_LSN
	-   POINT
	-   POLYGON
	-   TSQUERY
	-   TSVECTOR
	-   TXID_SNAPSHOT
